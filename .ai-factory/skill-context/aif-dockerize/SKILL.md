**Source**: 2026-06-21-13.05.md
**Rule**: When generating or auditing Compose profiles, ensure every profile gets distinct host port defaults. **Source**: 2026-06-24-18.30.md
**Rule**: For Docker init containers using Alpine, avoid runtime `apk add` for critical startup logic. Bake all required dependencies into the image. For template substitution, prefer `sed` over `envsubst` to avoid the `gettext` dependency on potentially EOL base images.
